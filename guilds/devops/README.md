# DevOps / Infrastructure Guild

## Overview

The DevOps / Infrastructure Guild focuses on building and maintaining the systems that enable reliable, scalable, and automated software delivery. Members work on CI/CD pipelines, cloud infrastructure, observability, and platform engineering.

---

## Focus Areas

- Continuous Integration and Continuous Deployment (CI/CD)
- Infrastructure as Code (IaC)
- Container orchestration and microservices infrastructure
- Monitoring, logging, and alerting
- Cloud architecture (multi-cloud)
- Security hardening and compliance automation
- Developer experience and platform tooling

## Standards

- All infrastructure must be defined as code. Manual configuration of production systems is prohibited.
- CI/CD pipelines must include linting, testing, and security scanning stages.
- Production deployments must support rollback mechanisms.
- All systems must have health check endpoints and alerting configured.
- Infrastructure changes must go through the same PR review process as application code.
