# Web Engineering Guild

## Overview

The Web Engineering Guild focuses on designing, building, and maintaining frontend systems, APIs, and full-stack platforms. Members of this guild work across the browser, server, and infrastructure layers of web applications.

---

## Focus Areas

- Frontend architecture and component systems
- RESTful and GraphQL API design
- Server-side rendering and static site generation
- Performance optimization and Core Web Vitals
- Accessibility compliance (WCAG 2.1 AA)
- Authentication and authorization systems

---

## Standards

- All components must be accessible and keyboard-navigable.
- APIs must follow RESTful conventions or GraphQL best practices.
- Frontend code must achieve a Lighthouse performance score of 90+.
- All user-facing text must support internationalization.
