# AI / Machine Learning Guild

## Overview

The AI / Machine Learning Guild focuses on building intelligent systems — from data pipelines and model training to deployment and monitoring. Members work across the full ML lifecycle, from research to production.

---

## Focus Areas

- Supervised and unsupervised learning models
- Natural language processing (NLP)
- Computer vision systems
- Data pipeline architecture and ETL processes
- Model serving and MLOps
- Experiment tracking and reproducibility

## Standards

- All models must have documented training procedures and evaluation metrics.
- Data pipelines must be reproducible and version-controlled.
- Model artifacts must be versioned and stored in a model registry.
- Production models must include monitoring for data drift and performance degradation.
- Experiments must be tracked with clear documentation of hyperparameters and results.
